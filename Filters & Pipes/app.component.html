Slice

property value | slice:start:end is used to slice a piece of data from an string {{value | slice:1:4}}

start and end are the starting and ending position. The index starts from 0. slice checks letters between (start,end-1) position.

Percent

property value | percent: 'minIntegerDigits.minFractionDigits-maxFractionDigits' is used to convert to percentage format {{n | percent:'2.2'}}

  • minIntegerDigits: minimum number of Integer digits.
  • minFractionDigits: minimum number of fraction digits.
  • maxFractionDigits: maximum number of fraction digits.
